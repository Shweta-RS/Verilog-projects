# Decoder Based RAM Design Using Verilog HDL

## Project Overview

This project implements a **Decoder Based RAM Architecture** using Verilog HDL.

The design uses multiple RAM blocks connected through a decoder logic to select individual memory modules based on address bits.

The project demonstrates:

- Memory organization using multiple RAM blocks
- Decoder based chip selection
- Read and write operations using tri-state buses
- Hierarchical hardware design using Verilog HDL
- Functional verification using testbench simulation

---

## Features

- 8-bit data width
- Address decoding for RAM selection
- Separate Read and Write operation
- Tri-state bidirectional data bus
- Synchronous memory operation
- Modular Verilog design
- Simulation verification included
