# 8×8 RAM Design Using Verilog HDL



## Project Overview



This project implements an 8×8 Random Access Memory (RAM) using Verilog HDL. The RAM supports synchronous write operations and read operations using separate read and write addresses.



The design includes RTL implementation and a testbench for functional verification through simulation.



---



## Features



* 8-bit data width

* 8 memory locations

* Separate read and write addressing

* Synchronous write operation

* Reset functionality

* Behavioral simulation verification
